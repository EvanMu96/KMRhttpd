# KMRhttpd
kqueue based C++ non-blocking http server on Unix.

## Usage
put your text file in the ./static directory.  
<b>to compile:</b>  
`
$ make  
`  
<b>to start a server:</b>  
`
$ ./KMRhttpd 8000 # 8000 as the listening port.
`
